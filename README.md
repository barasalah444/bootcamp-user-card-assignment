# Profile Card Component 🌐

A clean, responsive profile user card component built as part of a web development assignment to practice advanced CSS layouts, positioning, and styling.

## 🚀 Objective
The goal of this project was to recreate a specific profile layout from a design mockup using semantic HTML and CSS properties (like absolute positioning, flexbox, and border radius modifications).

## ✨ Features
* **User Card Layout:** Features a teal header banner with a profile picture overlapping it.
* **Overlapping Profile Image:** Achieved using precise CSS positioning (`position: relative` / `absolute` or negative margins).
* **Typography & Icons:** Clear hierarchy for user details (Name, Location, Profession, and Edit Profile link).
* **Fully Responsive:** Adapts cleanly to different screen dimensions.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Custom styling, layout management, and positioning.

## 📸 Mockup Reference
The design was built to match the following layout requirement:
* Teal-colored banner top.
* Circular avatar overlapping the banner boundary.
* Profile details left-aligned beneath the avatar.
